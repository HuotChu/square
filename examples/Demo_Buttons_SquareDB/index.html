Architecture

Before we can begin building an app, there needs to be a structure to follow so we know where everything will go.

For this demo, our structure looks like the following image. Replicate this structure by creating the required directories and empty files or download the demo from GitHub if you'd like to follow along.

The structure pictured is missing the square framework, because you can keep it anywhere you want. For this tutorial, create a new 'lib' directory in your 'Demo_Buttons_SquareDB' directory. Copy the contents of the SquareJS GitHub 'lib' directory into the 'lib' directory you just created. Now open up index.html and let's create a configuration to tell our app where it can find everything.
